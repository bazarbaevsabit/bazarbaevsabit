- 🤖 **Введение в ROS**  
  Практика работы с Robot Operating System 2, моделирование, навигация, компьютерное зрение.
- 🖥️ **Открытые платформы виртуализации**  
  Контейнеризация, Docker, Proxmox, KVM и облачные сервисы.

I'm Sabit Bazarbaev, a senior lecturer teaching robotics and virtualization courses. I help students master ROS, Docker, and adjacent technologies. This GitHub is an open collection of educational examples, lab assignments, and experimental playgrounds.

### 📚 Courses I teach

- 🤖 **Introduction to ROS**  
  Hands-on work with Robot Operating System 2: simulation, navigation, computer vision.
- 🖥️ **Open Virtualization Platforms**  
  Containerization, Docker, Proxmox, KVM, and cloud services.

### ⭐ Featured repositories

These are the projects we actively use in class.  
The full list is available in [repositories](https://github.com/bazarbaevsabit?tab=repositories).

| Project | Description | Stack |
|---------|-------------|-------|
| [color-follower-ros2](https://github.com/bazarbaevsabit/color-follower-ros2) | A robot that follows a colored object. Demonstrates image processing and control loops in ROS2. | ROS2 Humble, OpenCV, Docker |
| [worlds_and_robots](https://github.com/bazarbaevsabit/worlds_and_robots) | Educational worlds and robot models for Gazebo. Modular structure: each robot and world is a separate package. | ROS2 Jazzy, Gazebo, URDF/XACRO |
| [color-follower-ros2-v2](https://github.com/bazarbaevsabit/color-follower-ros2-v2) | Improved version of the color follower. | ROS2, Docker |

> 💡 **Tip for students:** start with [`worlds_and_robots`](https://github.com/bazarbaevsabit/worlds_and_robots) to understand the basic structure of ROS packages and how to work with the simulator.
