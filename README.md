# 👋 Привет! Меня зовут Сабит Базарбаев


**Старший преподаватель кафедры «Автоматизация и робототехника» (АРТ) ОмГТУ**

Веду курсы по робототехнике и виртуализации, помогаю студентам разбираться с ROS, Docker и смежными технологиями. Этот GitHub — открытая база учебных примеров, лабораторных работ и полигонов для экспериментов.

---

### 📚 Дисциплины, которые я веду

- 🤖 **Введение в ROS**  
  Практика работы с Robot Operating System 2, моделирование, навигация, компьютерное зрение.
- 🖥️ **Открытые платформы виртуализации**  
  Контейнеризация, Docker, Proxmox, KVM и облачные сервисы.
